How to view the preview pages (no server needed)
===============================================

1. In Finder, go to this folder:
   "Burning Man/web page for camp fly"

2. Make sure these items are in the same folder:
   - Camp-Fly-Home-Preview.html
   - Camp-Fly-Media-Preview.html
   - Camp-Fly-Preview.html
   - logo/   (folder with camp-fly-logo.png and dragonfly-only.png)
   - photos/ (folder with your art car images, for the home slideshow)

3. Double-click any of the HTML files. They will open in your default
   web browser (Safari, Chrome, etc.).

4. Use the links at the top to switch between Home, Burgin Initiation,
   and Media. On the Media page, enter the password and click
   "View media" to see the album cards.

5. To save a page as PDF: File → Print → Save as PDF (or use the
   print dialog’s "Save as PDF" option).

If the logo or photos don’t show up, check that the "logo" and "photos"
folders are in the same folder as the HTML files.
